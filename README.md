## ✦ Sobre o Projeto

Este é o portfólio profissional de **Gilcy Kelly dos Santos Rodrigues**, farmacêutica clínica atuante em Goiânia – GO. O site foi concebido com a proposta de apresentar a trajetória profissional com elegância, credibilidade e clareza, sem abrir mão de uma experiência visual sofisticada.

O design segue uma linha **premium minimalista**: tipografia editorial, paleta contida, espaço negativo generoso e microinterações suaves — tudo em um único arquivo HTML, sem dependências de frameworks pesados.

---

## ✦ Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| **HTML5 semântico** | Estrutura e acessibilidade |
| **CSS3 (Custom Properties)** | Design system, layout Grid & Flexbox |
| **Google Fonts** | Cormorant Garamond + DM Sans |
| **Vanilla JavaScript** | Intersection Observer (animações de scroll) |
| **GitHub Pages** | Hospedagem gratuita e contínua |

> Nenhuma biblioteca JavaScript externa. Zero dependências de build. Um único arquivo `index.html`.

## ✦ Como Usar Localmente

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/portfolio-gilcy.git

# 2. Acesse a pasta
cd portfolio-gilcy

# 3. Abra no navegador
# Basta abrir o arquivo index.html diretamente
open index.html         # macOS
start index.html        # Windows
xdg-open index.html     # Linux
```

Não é necessário nenhum servidor ou instalação adicional.

---

## ✦ Deploy com GitHub Pages

1. Acesse **Settings** → **Pages** no repositório
2. Em **Source**, selecione `Deploy from a branch`
3. Escolha a branch `main` e a pasta `/ (root)`
4. Clique em **Save**
5. O site ficará disponível em:
   `https://seu-usuario.github.io/portfolio-gilcy`
